Augments The Cheat Bunker with content from the Automatron DLC.

Quest Auto-completion options:
Unlock all the robot workbench parts as well as the Eyebot pod and the oscillating spotlight
Collect the holotapes for the alternative ending to the quest “Restoring Order” http://fallout.wikia.com/wiki/Restoring_Order
Importers: None
Exterior space connection(s): N/A

Items modified for playability and/or adding a legendary effect attach point:
Tesla Rifle
Mr. Handy Buzz Blade
Using a mod which edits these items will conflict with The Cheat Bunker: Automatron, but not in a way that prevents using both.

The Cheat Bunker: Automatron has been cleaned with FO4Edit.
